Cybersecurity Program Overview

Executive Summary - Critical Infrastructure Protection
HIGH-LEVEL OVERVIEW

Mission: Syntra Refining's Master Cybersecurity Program protects critical refining operations, safeguards proprietary technology, ensures regulatory compliance, and maintains stakeholder trust through comprehensive security controls and continuous monitoring.

2 Critical Asset Protection

4-Tier Asset Classification

  • Critical: SCADA, process controls, safety systems (air-gapped, continuous monitoring)
  • High: Proprietary refining processes, R&D data (encryption, DLP, strict access controls)
  • Medium: ERP, financial systems, HR databases (standard encryption, RBAC)
  • Standard: Email, collaboration tools, public website (basic security controls)

Network Segmentation

  • 6-layer architecture from process sensors to enterprise systems
  • Mandatory security inspection zones between OT and IT networks
  • Deep packet inspection and protocol filtering at all boundaries

3 Security Operations

<15 min Critical Incident Response
24/7 SOC Monitoring
<1 hr Safety System Recovery

Detection & Response Capabilities

  • SIEM Platform: Centralized log aggregation and threat correlation
  • Network Analysis: Anomaly detection and behavioral baseline monitoring
  • EDR Solutions: Host-based threat detection and automated response
  • OT Monitoring: Industrial protocol analysis and asset tracking
  • Threat Intelligence: Real-time APT feeds targeting critical infrastructure

4 Access Control & Identity

Multi-Factor Authentication
Required for all remote access and privileged accounts
Privileged Access Management
Just-in-time access with approval workflows and session recording
Biometric Authentication
Required for critical facility access and OT system modifications
Least Privilege
Default deny posture with quarterly access certification reviews

5 Training & Awareness

  • All Employees: Annual security awareness fundamentals (2 hours)
  • OT Personnel: Semi-annual ICS security training (4 hours)
  • IT Staff: Quarterly advanced security operations (8 hours)
  • Developers: Quarterly secure coding and DevSecOps (6 hours)
  • Executives: Annual cyber risk and governance (3 hours)

Continuous Testing

  • Monthly phishing simulations with immediate micro-training
  • Quarterly social engineering tests (phone, email, physical)
  • Executive impersonation scenario testing

6 Data Protection

Encryption Standards
AES-256 at rest, TLS 1.3 in transit for all data
Backup Strategy
3-2-1 backup with quarterly full restoration tests
IP Protection
Compartmentalized R&D access with DRM and watermarking
Data Loss Prevention
Automated scanning and blocking of unauthorized transfers

  • Contractual Controls: Security requirements, audit rights, incident notification, insurance
  • Ongoing Monitoring: Quarterly attestations, annual audits, continuous threat intelligence
  • Supply Chain Security: SBOM reviews, hardware integrity verification, trusted supplier programs
