## HERO SECTION ANIMATIONS

### Particle Backgrounds
**See it in action:**
- stripe.com - Homepage hero with subtle floating elements
- apple.com/airpods-pro - Product page with particle effects
- threejs.org - Interactive 3D particles

**What to notice:**
- Particles move slowly (almost imperceptible)
- Low opacity (10-20%)
- Responds to mouse movement
- Doesn't distract from content

### Typography Animations
**See it in action:**
- airbnb.com - Category page headers
- spotify.com/premium - Feature section text
- figma.com - Homepage headline

**What to notice:**
- Text appears letter-by-letter or word-by-word
- Timing feels natural (not too fast/slow)
- Only animates once on page load
- Uses subtle fade + slide

## ANALYZING A "WOW" WEBSITE

### Step-by-Step Analysis Method

When you find a website you like:

1. **First Impression** (5 seconds)
   - What catches your eye first?
   - How does the hero section feel?
   - Is the value proposition clear?

2. **Navigation** (15 seconds)
   - How does the menu work?
   - Can you find key pages quickly?
   - Is the mobile menu elegant?

3. **Scroll Experience** (30 seconds)
   - Do elements animate in?
   - Is there parallax?
   - Does anything feel laggy?

4. **Interactions** (1 minute)
   - Hover over buttons and cards
   - Try form fields if present
   - Click through a few pages
   - Note any delightful moments

5. **Details** (2 minutes)
   - Typography choices
   - Color transitions
   - Loading states
   - Error handling
   - Mobile responsiveness

6. **Performance** (Check with tools)
   - Open Chrome DevTools
   - Check Lighthouse score
   - Note page load time
   - See if animations are smooth

## QUICK TEST: IS AN EFFECT "WOW" OR "TOO MUCH"?

Ask yourself:

1. **Does it serve a purpose?**
   - ✓ Guides attention → Good
   - ✗ Just looks cool → Remove

2. **Is it subtle enough?**
   - ✓ You notice once → Good
   - ✗ You can't ignore it → Tone down

3. **Does it slow things down?**
   - ✓ Page still feels fast → Good
   - ✗ Delays content → Remove or async load

4. **Would you notice if it was gone?**
   - ✓ Yes, something feels missing → Keep
   - ✗ No difference → Remove

5. **Does it work on mobile?**
   - ✓ Yes, or has mobile alternative → Good
   - ✗ Broken/janky on mobile → Disable for mobile
