# Syntra GitHub Pages Publish Package

This package contains the approved public Syntra website pages only.

## Included pages

- Home: `index.html`
- Commercial: `commercial.html`
- Technology: `technology.html`
- Company: `company.html`
- Careers: `careers.html`
- Contact Us: `contact.html`
- Observer Portal: `observer.html`
- Supplier Portal: `supplier.html`
- Terms: `terms.html`
- Privacy: `privacy.html`

## Publishing steps

1. Unzip this package.
2. Replace the contents of your current GitHub Pages repository with these files.
3. Keep your existing `.git` folder if you are working locally.
4. Commit and push the update.
5. In GitHub Pages settings, publish from the branch root.

## Notes

- The `CNAME` file is included for `www.syntrarefining.com`.
- `.nojekyll` is included so GitHub Pages serves static assets as-is.
- Industry, production, internal admin, development, and experimental pages are intentionally excluded.
