Subtle motion effects (gradient shift + logo pulse). Conveys advanced technology. Falls back gracefully.

What's the difference?

Both versions have identical layout, content, and professionalism. The animated version adds barely-visible motion effects that suggest precision and advanced technology. Most organizations should start with the static version.

  • Static: 100% compatibility, zero risk
  • Animated: 85-90% compatibility, subtle tech positioning

Quick Start Guide

For IT Administrators:

  1. Download the Static Version (recommended)
  2. Download DEPLOYMENT_GUIDE.md and TEMPLATE_VARIABLES.md
  3. Host the Syntra logo on your web server (HTTPS required)
  4. Update the image URL in the signature HTML
  5. Map template variables to Exchange dynamic fields
  6. Create Exchange Transport Rule in Microsoft 365
  7. Test with a small group before organization-wide rollout

For Decision Makers:

  1. Download preview.html and open in browser
  2. Download SUMMARY.md for executive overview
  3. Download VERSION_COMPARISON.md to choose version
  4. Approve and delegate deployment to IT team
