### Why Use This?
- Fastest method for batch recording

### How to Use:

1. **Open the tool:**
   ```
   /email-signature/quick-batch-recorder.html
   ```

   - Standard (600x200) - Recommended
   - Large (800x300) - Higher quality

   - **Windows:** ScreenToGif (free)

   - Automatically advances to next signature
   - Repeat for all 15 signatures

6. **Save your recordings:**
   - Save each as GIF
   - Name them: `firstname-lastname-signature.gif`
   - Optimize for email (under 1MB)

## üí° Pro Tips

1. Record at larger size (1000x350 or 1200x400)
2. Use 25-30 FPS
3. Resize down to 600x200 after recording
4. This gives super crisp results!

4. Don't stop between signatures!

3. Use ezgif.com/optimize after

## üîß Troubleshooting

### Quick Batch Recorder Issues:

**Signature not loading?**
- Refresh the page
- Check that signature files exist
- Try a different browser

**Timer not working?**
- Click "START RECORDING" button
- Or press SPACEBAR key
- Make sure page has focus

**Recording quality poor?**
- Choose larger size (XL or XXL)
- Use better recording software
- Increase frame rate in recorder

### Automated Generator Issues:

**Generation stuck?**
- Keep browser tab active
- Don't switch to other tabs
- Check browser console for errors
- Try smaller size or lower quality

**File size too large?**
- Lower quality setting
- Reduce frame rate to 20 FPS
- Use ezgif.com to optimize after

**Browser crashes?**
- This tool is memory intensive
- Close other tabs
- Try generating in batches
- Use Quick Batch Recorder instead
